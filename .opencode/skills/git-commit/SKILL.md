---
name: git-commit
description: Git commit message convention with emoji and commitzen types
---

コミットメッセージは以下の形式に従う:

```
{emoji} {type}({scope}): {description}

- 箇条書きで変更内容
```

| type | 絵文字 | 用途 |
|------|--------|------|
| feat | ✨ | 新機能 |
| fix | 🐛 | バグ修正 |
| docs | 📝 | ドキュメント |
| build | 🏗️ | ビルドシステム |
| refactor | ♻️ | リファクタリング |
| chore | 🔧 | 雑務 |
| ci | 👷 | CI設定 |
| perf | ⚡️ | パフォーマンス改善 |
| test | ✅ | テスト追加・修正 |
| style | 🎨 | コードスタイル |
| revert | ⏪️ | 変更取り消し |
