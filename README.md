# holly/dotfiles

- .tmux.conf for [tmux](http://tmux.sourceforge.net/)
- .vimrc for [vim](http://www.vim.org/)
- .bashrc
- .gitconfig
- .gitignore, README
- symlink.sh as install script

## how to use
    $ cd
    $ git clone git://github.com/holly/dotfiles.git ~/.dotfiles
    $ sh .dotfiles/symlink.sh
